1）、MTA（邮件传输代理）：一款邮件服务器软件，例如：exchange、sendmail、Imail2）、MTU（邮件用户代理）：一款邮件客户端软件，例如：outlook、foxmail3）、MTX（邮件传输系统）：邮件服务器和邮件客户端组成的邮件传输体系4）、pop（邮局通信协议）：用于邮件接收（先下载，再删除）5）、imap（internet消息访问协议）：用于接收邮件（在线读取）6）、smtp（简单邮件传输协议）：用于发邮件

　一般邮件系统的角色结构可以分为三类： 
　　MTA:邮件传输代理 Mail Transfer Agent 
　　MDA:邮件递交代理 Mail Delivery Agent 
　　MUA:邮件用户代理 Mail User Agent
